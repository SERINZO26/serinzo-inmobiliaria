# Documentacion de la API

Base URL: `http://localhost:4000`

Todas las respuestas siguen el formato:
```json
{
  "success": true,
  "data": {},
  "error": null,
  "meta": {}
}
```

---

## Autenticacion

Todos los endpoints del panel requieren JWT en el header:
```
Authorization: Bearer <token>
```
